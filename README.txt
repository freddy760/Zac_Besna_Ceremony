BESNA CEREMONY — Photo Slideshow
================================

HOW TO RUN
  1. Keep all files in this folder together.
  2. Double-click index.html to open it in Chrome.
     (For best results, see the sharing options in the chat instructions.)

CONTROLS
  - Play / Pause button  (or press the Spacebar)
  - Previous / Next       (or the Left / Right arrow keys)
  - Swipe left / right on phones and tablets
  - "Play Soft Music" to start optional looping music; then Mute/Unmute appears

ADD MUSIC (optional)
  - Put a file named  soft-music.mp3  inside the  audio  folder.
  - Use royalty-free or your own personal music only.

EDIT PHOTOS OR CAPTIONS
  - Photos live in the  images  folder, named 01.jpg ... 32.jpg (in order).
  - To change captions, open script.js and edit the `caption:` text.
